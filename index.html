A propos de moi:


Bonjour, je suis Kuro !
Je suis un lycéen Français, j'ai commencé à programmer il y a quelques années déjà,
mais j'ai vraiment commencé en entrant au lycée.
J'ai commencé avec le Python (que j'apprécie beaucoup), puis je me suis essayé à l'HTML / CSS / JS
que j'ai un peu moins travaillé mais suffisament pour les utiliser pour l'application mobile
pour mon projet de terminale. J'ai aussi vu les bases du PHP, SQL, Ruby, C / C++...
Et plus récemment, je m'essaye au Rust et au Lua (avec le game engine LÖVE).

Je possède aussi un RaspberryPi qui me sert de serveur de test pour mes différents projets
nécessitants une BDD avec un apache. Il me sert aussi pour héberger un Cloud personnel et un serveur mail.

Si vous voulez savoir quelque chose, n'hésitez pas à demander ! o/ Je ne mords pas ;)
